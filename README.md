# BAO 30k Submission

This is my submission for the Bedrock Add-Ons Server 30K jam!

- Mine corruption - Turns nearby blocks into stone and ores!

- Orbital Laser - Uhhh digs a hole lol
- Jebsplosion - A rainbow explosion
- Danger rain - Better take cover from this one

- Villager - Just a regular toolsmith
